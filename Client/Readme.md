# Device Part 
### 파일 & 폴더 설명
- models : 객체 탐지 모델 & dlib 관련 파일 폴더
- Client.py : 실제로 구동 시키는 파일
- Detect.py : 마스크 미착용자 탐지 관련 클래스
- Recognition.py : 얼굴 인식 관련 클래스
- Session.py : 인증 관련 클래스
- Only_Detect.py : 마스크 미착용자 탐지 기능만 작동하는 코드 

### 필요 패키지
- dlib
- opencv-python
- requests
- 나머지는 requirements.txt 참고
